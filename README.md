# 🛒 Amazon Clone

A front-end e-commerce web application built using HTML, CSS, and JavaScript that replicates the look and basic functionality of the Amazon website.

---

## 📋 Project Description

This project is a simplified version of the Amazon website designed to demonstrate core web development concepts. It includes product listings, UI components, and interactive features to enhance user experience.

---

## 🚀 Features

* 🛍️ Product listing interface
* 🧺 Shopping cart UI
* 🔍 Search bar design
* 🎨 Responsive and attractive layout
* ⚡ Interactive elements using JavaScript

---

## 🛠️ Technology Stack

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity and dynamic behavior

---
